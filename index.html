Zalo System Admin Dashboard

Project :Admin Dashboard

Client :Zalo

Technology :Java/Jetty/NoSQL

Preview : Private Project

Project description:

This project is used to visualize statistics of DBM, message broker services and some internal data, make some alert notifications and change some config with GUI. In this project, I tried to collect the high volume of data from message broker services and save them as time series DB in an efficient way, with durable and minimal data storage.

Summarize of the implemented feature:

  • Implement the agent service to collect the data from message brokers.
  • Store the data in an efficient way in DB. Expose the APIs for time-series data to FE.
  • Implement the mechanism for anomaly detection, send alerts to proper PIC.
  • Expose some APIs to manage mesage brockers.
